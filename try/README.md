# Guida Try

This is an example of how to use the browser version of the compiler.

## How to run

To run this example, follow these steps:

1. Start by building guida API at the top level folder by running the following:

```
nvm use
npm install
npm run build:api
```

2. Move into this folder and install the dependecies:

```
cd try
npm install
```

3. Start the server with `npm start`
4. Open http://127.0.0.1:8088
